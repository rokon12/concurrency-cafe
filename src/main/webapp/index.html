The shared variables for this level are typed. Operations must match the declared type — the parser will tell you if you mismatch.

  • Thread.ofVirtual().start(() -> { ... }); — start a virtual thread chef
  • Thread.ofPlatform().start(() -> { ... }); — start a platform thread chef
  • int: int x = v;, v = expr;, v++, v += n
  • AtomicInteger: v.incrementAndGet();, v.set(n);
  • Object monitor: synchronized (m) { ... }
  • ReentrantLock: l.lock(); / l.unlock();
  • System.out.println("..."); — emit a log line
